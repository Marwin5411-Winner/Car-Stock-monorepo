## Top Priority Fix List

The following 5 bugs should be fixed immediately as they represent active security vulnerabilities or data corruption risks:

1. **#1 — Hardcoded Discord webhook** — rotate the exposed webhook URL and remove the fallback
2. **#2 — Privilege escalation** — any user can make themselves admin via self-update
3. **#4 — JWT refresh breaks auth** — all users who refresh their token lose authentication
4. **#5 — Unauthenticated payment report** — financial data exposed without login
5. **#7-9 — Non-atomic financial operations** — debt payments and sales can corrupt data under concurrent access
